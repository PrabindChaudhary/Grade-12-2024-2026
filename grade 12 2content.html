Database Management System

A Database Management System (DBMS) is software that allows users to define, create, maintain, and control access to the database. It helps in managing large amounts of data efficiently and ensures data integrity, security, and consistency.

  • Data vs Information: Data is raw facts; information is processed data.
  • Features of Information: Timeliness, Accuracy, Relevance, Completeness
  • Database Purpose: Efficient data storage and retrieval
  • Terminologies: Table, Field, Record, Tuple, Object, Keys
  • Data Dictionary: Metadata repository
  • Objectives of DBMS: Data integrity, security, sharing, backup
  • Advantages: Reduced redundancy, consistency, security
  • Disadvantages: Cost, complexity, size

Data Communication and Networking

This topic covers how data is transmitted between computers and devices using transmission mediums. It includes network topologies, protocols, OSI and TCP/IP models, and networking hardware.

Object Oriented Programming

OOP is a paradigm based on the concept of "objects" which contain data and code. It covers classes, inheritance, polymorphism, encapsulation, abstraction, and object-oriented design principles.

Software Development Life Cycle

SDLC is a process for planning, creating, testing, and deploying information systems. Phases include Requirement Gathering, Design, Implementation, Testing, Deployment, and Maintenance.
